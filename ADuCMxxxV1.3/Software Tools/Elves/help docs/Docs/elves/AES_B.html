Help Setup Context

Pressing the Setup button opens the setup window as shown:

In the Browser edit box type the full path and application name of the browser you would like to use for help or select it using the browse feature.

In the Help Path edit box type the full path to the help files. If you do not know where they are search for AE_P.html and use its path or use the browse feature.

Select the Details radio button to get the help as selected above or the Intro radio button to get the introductory help file in Notepad instead.

Press the OK button to fix these settings or Cancel to discard the changes.
