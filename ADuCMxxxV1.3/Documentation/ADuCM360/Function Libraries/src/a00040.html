Detailed Description

Set of DAC peripheral functions.

  • First configure DAC with DacCfg().
  • Set sync mode with DacSyn() if desired.
  • Output DAC value with DacWr().
  • Example: DacCfg(0,DAC_REF_INT,DAC_VOUT,DAC_12BIT); for(i1 = 0; i1<0x10000000; i1 += 0x1000000) DacWr(0,i1);
