int UrtBrk(ADI_UART_TypeDef *pPort, int iBrk) ==========Force SOUT pin to 0

Parameters
pPort:{pADI_UART,}
Set to pADI_UART. Only one channel available.
iBrk:{COMLCR_BRK_CLR, COMLCR_BRK_SET}
  • 0 = COMLCR_BRK_CLR to disable SOUT break condition (SOUT behaves as normal)
  • 1 = COMLCR_BRK_SET to force SOUT break condition - SOUT remains low until this bit is cleared
Returns

int UrtCfg(ADI_UART_TypeDef *pPort, int iBaud, int iBits, int iFormat) ==========Configure the UART.

Parameters
pPort:{pADI_UART,}
Set to pADI_UART. Only one channel available.
iBaud:{B1200,B2200,B2400,B4800,B9600,B19200,B38400,B57600,B115200,B230400,B430800}
Set iBaud to the baudrate required: Values usually: 1200, 2200 (for HART), 2400, 4800, 9600, 19200, 38400, 57600, 115200, 230400, 430800, or type in baud-rate directly
iBits:{COMLCR_WLS_5BITS,COMLCR_WLS_6BITS,COMLCR_WLS_7BITS,COMLCR_WLS_8BITS}
  • 0 = COMLCR_WLS_5BITS for data length 5bits.
  • 1 = COMLCR_WLS_6BITS for data length 6bits.
  • 2 = COMLCR_WLS_7BITS for data length 7bits.
  • 3 = COMLCR_WLS_8BITS for data length 8bits.
iFormat:{0|COMLCR_STOP_EN|COMLCR_PEN_EN|COMLCR_EPS_EN|COMLCR_SP_EN}
The bitwise or of the data format required:
  • 0 = URT_STP1 -> 1 stop bits, no parity.
  • 0x4 = COMLCR_STOP_EN -> Set for 2 stop bits.
  • 0x8 = COMLCR_PEN_EN -> Set for Parity enabled
  • 0x10 = COMLCR_EPS_EN -> Even parity.
  • 0x20 = COMLCR_SP_EN -> Sticky parity.
Returns
Value of COMLSR: See UrtLinSta() function for bit details.
Note
  • Powers up UART if not powered up.
  • Standard baudrates are accurate to better than 0.1% plus clock error.
  • Non standard baudrates are accurate to better than 1% plus clock error.
Examples:
ADC0_DMA.c, ADC1_DMA.c, ADCMeter.c, RTD_Demo.c, RTD_toSpi.c, Thermocouple_to_DAC.c, Thermocouple_to_UART.c, UARTDma_Demo.c, and UartDmaTx.c.

int UrtIntCfg(ADI_UART_TypeDef *pPort, int iIrq) ==========Enables/Disables UART Interrupt sources.

Parameters
pPort:{pADI_UART,}
Set to pADI_UART. Only one channel available.
iIrq:{COMIEN_ERBFI| COMIEN_ETBEI| COMIEN_ELSI| COMIEN_EDSSI| COMIEN_EDMAT| COMIEN_EDMAR}
  • 0 to select none of the options. Or set to the bitwise or combination of
  • COMIEN_ERBFI to enable UART RX IRQ.

  • COMIEN_EDSSI to enable UART Modem status IRQ.
  • COMIEN_EDMAT to enable UART DMA Tx IRQ.
  • COMIEN_EDMAR to enable UART DMA Rx IRQ.
Returns
1.

int UrtMod(ADI_UART_TypeDef *pPort, int iMcr, int iWr) ==========Write iMcr to UART Modem Control Register

Parameters
pPort:{pADI_UART,}
Set to pADI_UART. Only one channel available.
iMcr:{0|COMMCR_DTR|COMMCR_RTS|COMMCR_LOOPBACK}
Set to the modem control combination required (COMMCR):
  • 0 to not use DTR, RTS ot LOOPBACK. Or a combination of the following.
  • COMMCR_DTR to use Data terminal ready.
  • COMMCR_RTS_to use Request to send.
  • COMMCR_LOOPBACK for Loop back mode.
iWr:{0,1}
  • 0 to read mode only (ignores iMcr).
  • 1 to write and read mode.
Returns
value of COMMSR:
  • COMMSR.0 = DCTS -> CTS changed.
  • COMMSR.1 = DDSR -> DSR changed.
  • COMMSR.2 = TERI -> RI Trailing edge.
  • COMMSR.3 = DDCD -> DCD changed.
  • COMMSR.4 = CTS -> Clear to send.
  • COMMSR.5 = DSR -> Data set ready.
  • COMMSR.6 = RI -> Ring indicator.
  • COMMSR.7 = DCD -> Data carrier detect.
Note
This function does not change the Port Multiplexers.
Examples:
ADC0_DMA.c, ADC1_DMA.c, ADCMeter.c, RTD_Demo.c, RTD_toSpi.c, Thermocouple_to_DAC.c, Thermocouple_to_UART.c, UARTDma_Demo.c, and UartDmaTx.c.

int UrtModSta(ADI_UART_TypeDef *pPort) ==========Read the Modem status register byte of the UART.

Parameters
pPort:{pADI_UART,}
Set to pADI_UART. Only one channel available.
Returns
value of COMMSR:
  • COMMSR_DCD = Data carrier detect level
  • COMMSR_RI = Ring indicator level
  • COMMSR_DSR = Data set ready status
  • COMMSR_CTS = Clear to Send input level
  • COMMSR_DDCD = Delta DCD status
  • COMMSR_TERI = trailing edge Ring indicator status
  • COMMSR_DDSR = Delta DSR status
  • COMMSR_DCTS = Delta CTS status
Warning
UART must be configured before checking status
