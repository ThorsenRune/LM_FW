int WdtCfg(int iPre, int iInt, int iPd); ========== Configures the watchdog timer.

Parameters
iPre:{T3CON_PRE_DIV1,T3CON_PRE_DIV16,T3CON_PRE_DIV256,T3CON_PRE_DIV4096}
  • T3CON_PRE_DIV1, for a prescaler of 1.
  • T3CON_PRE_DIV16, for a prescaler of 16.
  • T3CON_PRE_DIV256, for a prescaler of 256.
  • T3CON_PRE_DIV4096, for a prescaler of 4096.
iInt:{T3CON_IRQ_DIS,T3CON_IRQ_EN}
  • T3CON_IRQ_DIS, to generate a reset on a timer timeout.
  • T3CON_IRQ_EN, to generate an interrupt on a timer timeout.
iPd:{T3CON_PD_DIS,T3CON_PD_EN}
  • T3CON_PD_DIS, the timer continues to count when in hibernate mode.
  • T3CON_PD_EN, the timer stops counting when in hibernate mode.
Returns
1.
Examples:
ADC0_DMA.c, ADC1_DMA.c, ADCMeter.c, Blink.c, DAC_DMA.c, FlashProtect.c, FlashSign.c, FlashWrite.c, I2CDMA_Master.c, I2CDMA_Slave.c, PowerDown.c, PWM.c, RTD_Demo.c, RTD_toSpi.c, SPIDMA_Master.c, SPIDMA_Slave.c, Thermocouple_to_DAC.c, Thermocouple_to_UART.c, Timers.c, UARTDma_Demo.c, and UartDmaTx.c.
